Manipulación de Datos con LocalStorage

LocalStorage es una herramienta compatible con HTML5 y los navegeradores, gracias a esta podemos almacenar de manera local datos en el navegador que utiliza el usuario mientras navega. Esta información quedará almacenada para que siempre que el usuario inicia en su navegador estos datos esten ahí. LocalStorage trabaja de manera similar que las cookies, sin embargo, cuenta con un espacio de 5mb y al trabajar de manera local no necesita hacer peticiones al servidor.

¿Como se utiliza LocalStorage?

LocalStorage es una herramienta que se utiliza via JavaScript, donde podremos manipular datos almacenados de acuerdo a los codigos programados con este lenguaje. Para acceder a este almacemaniento se hace: Una vez ejecutada la aplicación--> Clic derecho-inspeccionar, vamos al apartado de Application(aplicación) y a mano izquierda aparece el menu del LocalStorage.

Metodos del LocalStorage para manipular datos

1.localStorage.setItem("clave",valor): Este es el metodo para almacenar información en nuestro LocalStorage. En el primer parametro "clave" es donde ingresamos el nombre del elemento y en el parametro valor ingresamos el dato que almacenará el parametro clave.

localStorage.getItem("clave"):Este metodo me sirve para traer el valor almacenado en una clave especifica

Intercambio de datos por medio de JSON

Para poder almacenar arreglos y objetos en el LocalStorage se deben de convertir en una cadena de Caracteres(String), para eso usaremos JSON.
JSON.stringify()

Convetir un String almacenado en el LocalStorage en un objeto: JSON.parse

Eliminar valores del LocalStorage

Se usa el metodo removeItem('') y recibe la clave del elemento que deseamos
