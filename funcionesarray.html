Funciones para la manipulacion de Array

1.¿como podemos aobtener la cantidad de elementos de una arreglo?

length: es la longitud del arreglo, quiere decir que devuelve el tamaño(cantidad) de datos que esta almacenado en un arreglo

2.¿como puedo remover un valor al final del arreglo

pop(): este metodo elimina el ultimo elemnto del arreglo.

3.¿como podemos agregar un nuevo valor al final del arreglo?

push(): ingresa un nuevo valor al final del arreglo.

4.¿como podemos eliminar el primer dato del arreglo?

shift(): elimina el primer valor del arreglo, se modifican los indices de ya que estamos cambiando el indice 0

5.¿como podemos agregar un valor al comienzo del arreglo?

unshift(): agrega un nuevo valor al comienzo del arreglo

6.¿como podemos eliminar y agregar valores al mismo tiempo?

splice(0.0,""): el primer 0 indica el indice donde va a empezar el cambio, el segundo 0 indica la cantidad de elementos que deseamos eliminar desde el indice que se indico anteriormente y las comillas es el nuevo valor que se agregara desde el indice indicado en el primer 0

7.¿como podemos convertir un arreglo en una cadena de texto?

.toString(): convierte la informacion del arreglo en una decadena de texto cada dato por comas(,)

8.¿Como podemos separar el arreglo en una dena de texto y definir su separador?

join(): este metodo tambien convierte la infomracion del arreglo en una cadena de texto pero le podemos cambiar las "," por otros separador?
