Manicupalcion de los datos con local Storage

localStroage es una herramienta comatible con HTML y los navegadores, gracias a esta podemos almacenar de manera local datos en el navegador que utiliza el usuariomientras navega. Esta infromacion quedara almacena para que siempre que le usuario inicia en su bavedor estos datos eten ahi, localStroage trabja de manera similar que las cookies, sub embargo, cuenta con un espacio de 5mb y altrabajar de manera local no necesita hacer peticiones al servidor.

Metodos del localStroage para manipular datos

1. localStroage.setItem("clave", valor): este es el metodo para almacenar informacion en nuetro localStorage. En el primer parametro "clave" es donde ingresamos el nombre del elemnetos y en el parametro valor, ingresamos el dato que almacenara el para metro clave.

2. localStorage.getItem("clave"): este metodo nos permite obtener el valor de un elemento almacenado de una clave especifica en el localStorage.

Intercambio de datos por medio de JSON

PAra pode almacenar arreglos y objetos en el LocalStorage se deben de convertir en una cadena de caracteres (String), para eso usaremos JSON.
Este metodo se cono como: JSON.strinify()

Convertir un string almca en el LOcalStorage en un objeto: JSON.parse
